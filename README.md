<h1 align="center">Remote Work tracking app in Flutter</h1>

<h2 align="center">
 The remote Work tracking app is a Full-fledged task tracker app developed in Flutter and Firebase.
</h2>

With this pack, you will get a suite of 2 Apps:
1. Employer app
2. Employee App

<h1 align="center">Features</h1>


1. Checkin and Check out
2. Create new tasks
3. Assign tasks to projects
4. Location tracking
5. Project creation
6. Full in house Calendar
7. Schedule events and meetings
8. Holiday and Task highlights in the calendar
9. Employee management system
10. Employee performance charts
11. Profile
12. Authentication by Firebase
13. Database by Firebase Firestore
14. Firebase notifications
15. Custom Color selection for each card/list
